Iroha.DateSelect

  • Necomesi JavaScript Library (Iroha)

概要

年/月/日の最大 3 つの select 要素を使って日付を選択する。

選択した年と月に存在しない日は自動的に選択肢から外される。

選択可能な日付の期間を範囲指定可能。

動作確認ブラウザ

(fixme)

実装ファイル配置

主ファイル
  • iroha.dateSelect.js
依存ファイル（主ファイルの動作に必要）
  • jQuery(jquery.js)(1.7.2+)
  • Iroha(iroha.js)(3.0+)

HTML への組み込み

<head>
    :
    <script type="text/javascript" src="path/to/jquery.js"></script>
    <script type="text/javascript" src="path/to/iroha.js"></script>
    <script type="text/javascript" src="path/to/iroha.dateSelect.js"></script>
    :
</head>
  • jquery.js, iroha.js は、これらに依存する各種スクリプトすべてより前に読み込ませておく必要があります。
