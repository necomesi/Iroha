Iroha 基本ライブラリ "iroha.js"

  • Necomesi JavaScript Library (Iroha)

概要

ほとんどの Iroha 系スクリプトが参照している基本ライブラリ。 jquery.js と合わせて使用します。

動作確認ブラウザ

(fixme)

実装ファイル配置

主ファイル
  • bajl.js
依存ファイル（主ファイルの動作に必要）
  • jQuery(jquery.js)(1.4.2+)

HTML への組み込み

<head>
    :
    <script type="text/javascript" src="path/to/jquery.js"></script>
    <script type="text/javascript" src="path/to/iroha.js"></script>
    :
</head>
  • jquery.js は、 iroha.js より前に読み込ませておく必要があります。
  • iroha.js は、これに依存している各種スクリプトすべてより前に読み込ませておく必要があります。
