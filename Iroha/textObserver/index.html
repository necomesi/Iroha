Iroha.TextObserver

  • Necomesi JavaScript Library (Iroha)

概要

任意要素内のテキストを監視し、変更された事をリアルタイムに検出します。とりわけ <input type="text"> や <textarea> の入力文字列変更をリアルタイムに検出したり、入力文字列の妥当性検証をリアルタイムに行ったりすることができます。
